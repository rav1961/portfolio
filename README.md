## Projects

### Product Configuration Platform

**Status:** Currently in development
**Source code:** Public 
**Repository:** https://github.com/rav1961/product-configurator
**Type:** Fullstack application / product configurator / quotation request system

**Backend:** PHP 8.4, Laravel 12, PostgreSQL, Redis, RabbitMQ, FilamentPHP
**Frontend:** Next.js 16, React 19, TypeScript

**Scope of work:**
A fullstack application consisting of a separate backend and frontend. The backend is responsible for business logic, product management, configuration rules, users, orders and the administration panel. The frontend is responsible for the customer interface, product configuration process and quotation request submission.
